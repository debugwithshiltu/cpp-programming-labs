# C++ Programming Labs 💻

Welcome to the **C++ Programming Labs** repository! This repository is a comprehensive, hands-on guide to learning C++ and mastering the **Standard Template Library (STL)**. 

Each folder contains specific C++ files (`.cpp`) dedicated to a core programming concept, data structure, or algorithm. This makes it easy for anyone to navigate, learn, and test the code step-by-step.

---

## 🗂️ Detailed Directory Structure & File Contents

Here is a complete breakdown of every folder and the code concepts you will find inside them.

### 🟢 1. Basics & Functions
**`0000_lab_intro_code`**
This folder covers the absolute basics of C++ syntax and structure.
* `hello_world.cpp` - Basic boilerplate code and `std::cout` usage.
* `basic_io.cpp` - Taking user input using `std::cin`.
* `data_types.cpp` - Understanding integers, floats, chars, and booleans.

**`010_Lab_of_using_function_use`**
Focuses on creating reusable blocks of code.
* `basic_functions.cpp` - Function declarations, definitions, and calling.
* `pass_by_value_ref.cpp` - Demonstrates passing parameters by value vs. passing by reference (`&`).
* `function_overloading.cpp` - Creating multiple functions with the same name but different parameters.

### 🟡 2. STL Fundamentals & Dynamic Arrays
**`000_STL(Standard Template Libary)`**
* `stl_intro.cpp` - High-level overview and basic usage of STL containers and iterators.

**`012_Lab_of_Vector_use`**
Deep dive into dynamic arrays (`std::vector`).
* `vector_basics.cpp` - Initialization, `push_back()`, `pop_back()`, and `size()`.
* `vector_traversal.cpp` - Iterating through vectors using index and range-based for loops.
* `vector_methods.cpp` - Using functions like `clear()`, `empty()`, and `erase()`.

### 🔵 3. Data Structures: Arrays & Strings
**`001_Lab_of_Array_use`**
* `array_traversal.cpp` - Iterating through fixed-size static arrays.
* `find_max_min.cpp` - Logic to find the largest and smallest numbers in an array.
* `reverse_array.cpp` - Reversing array elements in place.

**`008_Lab_of_String_use`**
* `string_operations.cpp` - Concatenation, length checking, and basic manipulation.
* `substrings.cpp` - Using the `substr()` function to extract parts of a string.
* `palindrome_check.cpp` - Algorithm to check if a string reads the same forwards and backwards.

### 🟠 4. Searching & Recursion Algorithms
**`002_Lab_of_Binary_Search_use`**
* `binary_search_iterative.cpp` - Finding elements in a sorted array using an iterative `while` loop.
* `binary_search_recursive.cpp` - Implementing binary search using recursive function calls.

**`006_Lab_of_Recurtion_use`**
* `factorial.cpp` - Calculating factorials using base cases and recursive calls.
* `fibonacci.cpp` - Generating the Fibonacci sequence recursively.

### 🟣 5. Sets, Multi-Sets, & Set Operations
**`007_Lab_of_Set_use`**
Working with `std::set` (stores ordered, unique elements).
* `set_basics.cpp` - Inserting elements and printing them (automatically sorted).
* `set_search_erase.cpp` - Using `find()` and `erase()` functions efficiently.

**`004_Lab_of_Multi_set_use`**
Working with `std::multiset` (stores ordered elements, duplicates allowed).
* `multiset_operations.cpp` - Inserting duplicate values and using the `count()` function.

**`003_Lab_of_Intersection_use`**
* `array_intersection.cpp` - Finding common elements between two distinct arrays using loops or pointers.
* `set_intersection.cpp` - Using STL algorithms (`std::set_intersection`) to find common elements.

**`009_Lab_of_Union_use`**
* `array_union.cpp` - Merging two arrays while ignoring duplicate values.
* `set_union.cpp` - Utilizing the nature of sets to easily combine datasets uniquely.

### ⚪ 6. Practice Labs (`Labs_001` to `Labs_012`)
These folders contain mixed practice problems, assignments, and mini-projects that combine multiple concepts from above.
* **`Labs_001` to `Labs_005`**: Early-stage practice focusing on arrays, loops, and basic functions.
* **`Labs_006` to `Labs_009`**: Intermediate practice focusing on strings, recursion, and sets.
* **`Labs_010` to `Labs_012`**: Advanced combinations of STL algorithms, vectors, and mathematical set operations.

## 🚀 How to Run the Code

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/debugwithshiltu/cpp-programming-labs.git](https://github.com/debugwithshiltu/cpp-programming-labs.git)

```

2. **Navigate into a specific folder:**
```bash
cd 001_Lab_of_Array_use

```


3. **Compile the C++ file** (Using G++ compiler):
```bash
g++ -o output_name filename.cpp

```


4. **Run the executable:**
* **Windows:** `output_name.exe`
* **macOS/Linux:** `./output_name`
